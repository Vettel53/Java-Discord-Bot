# **Fortnite Stats Discord Bot**
A Discord bot that retrieves and displays Fortnite player statistics using the Fortnite API. The bot is designed to parse JSON data, format it into a clean output, and send the player stats in response to user commands.

# **Features**
 * Retrieves Fortnite player stats (kills, wins, playtime, K/D ratio, etc.)
 * Fetches real-time data via the Fortnite API
 * Parses JSON responses and structures data in the PlayerStats model
 * Supports slash commands for user interaction in Discord
 * Handles optional parameters like account type, time window, and image via API calls
